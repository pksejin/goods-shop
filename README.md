# 잔재미코딩 굿즈샵

잔재미코딩 공식 굿즈를 판매하는 쇼핑몰입니다.

## 테스트 계정

| 구분 | 이메일 | 비밀번호 |
|---|---|---|
| 일반 회원 | 직접 회원가입 | 6자 이상 |
| 관리자 | admin@admin.com | superadmin |

## 페이지 목록

| URL | 설명 |
|---|---|
| `/index.html` | 상품 목록 |
| `/login.html` | 로그인 |
| `/signup.html` | 회원가입 |
| `/cart.html` | 장바구니 |
| `/orders.html` | 내 주문 내역 |
| `/admin.html` | 관리자 페이지 |

## 결제 테스트

토스페이먼츠 테스트 모드로 동작합니다.
테스트 카드 번호: `4242424242424242` (유효기간, CVC 임의 입력)

## 기술 스택

- **프론트엔드**: HTML / CSS / JavaScript (빌드 도구 없음)
- **인증·DB**: Supabase
- **결제**: 토스페이먼츠 (테스트 모드)
- **호스팅**: GitHub Pages
